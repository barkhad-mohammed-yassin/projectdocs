Week 4 - HTML, CSS & Publicatievoorbereiding

In week vier hebben we de overstap gemaakt van het ontwerp naar de technische kant van het project. We hebben een GitHub-account aangemaakt om onze bestanden online op te slaan en te delen, en een Netlify-account om onze website te kunnen publiceren.

Daarna zijn we aan de slag gegaan met HTML en CSS in Visual Studio Code. Ik leerde hoe ik de basisstructuur van een webpagina opbouw met HTML en hoe ik met CSS de tekst en lay-out kan aanpassen. Het was interessant om mijn ontwerp langzaam tot leven te zien komen in code. Deze week vormde de basis voor het technisch deel van mijn projectdocumentatie.
